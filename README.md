# Todo List 管理系统

> 基于 Vue3 + Element Plus + Express + Sequelize 开发的后台用户管理系统，支持用户登录、权限校验、用户 CRUD 及状态管理。

---

## 📌 项目简介
本项目是一个前后端分离的Todo List管理后台，核心功能包含：
- 用户登录/注册
- JWT 权限校验
- 用户列表分页查询
- 用户新增、编辑、删除
- 用户启用/禁用状态切换
- 表单校验与错误提示
- 响应式布局与基础权限控制

---

## 🛠 技术栈

### 前端
- **框架**: Vue 3 (Composition API)
- **UI 组件库**: Element Plus
- **路由**: Vue Router 4
- **状态管理**: Pinia
- **HTTP 请求**: Axios
- **日期处理**: dayjs
- **代码格式化**: Prettier

### 后端
- **运行环境**: Node.js
- **框架**: Express
- **ORM**: Sequelize (MySQL)
- **身份认证**: JWT
- **密码加密**: bcrypt
- **加密工具**: crypto
